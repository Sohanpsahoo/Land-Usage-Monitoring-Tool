Land Classification AI Model

Advanced machine learning model for satellite imagery classification

About the Model

Our land classification model is built using a Convolutional Neural Network (CNN) architecture, specifically designed to analyze satellite imagery. The model can classify land into 10 different categories:

  • Annual Crop
  • Forest
  • Herbaceous Vegetation
  • Highway
  • Industrial
  • Pasture
  • Permanent Crop
  • Residential
  • River
  • Sea/Lake

Model Architecture

The model consists of:

  • 3 Convolutional layers with ReLU activation
  • 2 MaxPooling layers
  • 1 Flatten layer
  • 2 Dense layers (64 and 10 neurons)
  • Softmax activation for classification
